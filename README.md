# **MERN TO-DO App**

## Run Backend

- `cd backend`
- Copy content from `.env.example` to `.env`
- `npm install`
- `npm run start` or `npm run dev` (hot reload)

## Run Frontend

- cd `frontend`
- Copy content from `.env.example` to `.env`
- `npm install`
- `npm run start`

Open your browser and go to `http://localhost:3000`
